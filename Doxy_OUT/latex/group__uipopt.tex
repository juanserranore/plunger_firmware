The netmask, default router and Ethernet address are appliciable only if u\+IP should be run over Ethernet.

\doxysubsection{Detailed Description}
u\+IP is configured using the per-\/project configuration file \doxyref{uipopt.\+h}{p.}{uipopt_8h}. This file contains all compile-\/time options for u\+IP and should be tweaked to match each specific project. The u\+IP distribution contains a documented example "{}uipopt.\+h"{} that can be copied and modified for each project.

\begin{DoxyNote}{Note}
Most of the configuration options in the \doxyref{uipopt.\+h}{p.}{uipopt_8h} should not be changed, but rather the per-\/project \doxyref{uip-\/conf.\+h}{p.}{uip-conf_8h_source} file. 
\end{DoxyNote}
