\doxysubsection{Detailed Description}
Representation of a u\+IP TCP connection.

The \doxyref{uip\+\_\+conn}{p.}{structuip__conn} structure is used for identifying a connection. All but one field in the structure are to be considered read-\/only by an application. The only exception is the appstate field whos purpose is to let the application store application-\/specific state (e.\+g., file pointers) for the connection. The type of this field is configured in the "{}uipopt.\+h"{} header file.
