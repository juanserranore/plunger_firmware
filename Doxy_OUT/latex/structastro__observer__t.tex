\doxysubsection{Detailed Description}
The location of an observer on (or near) the surface of the Earth. 

This structure is passed to functions that calculate phenomena as observed from a particular place on the Earth.

You can create this structure directly, or you can call the convenience function Astronomy\+\_\+\+Make\+Observer\# to create one for you.
