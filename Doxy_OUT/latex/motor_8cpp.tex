Moves the stepper motor a specified number of steps. 

This function controls the stepper motor by setting the direction pin and generating step pulses to move the motor. It updates the motor\textquotesingle{}s position and refreshes the screen if the current menu is MANUAL or AUTO.


\begin{DoxyParams}{Parameters}
{\em steps} & The number of steps to move the motor. Positive values move the motor in one direction, and negative values move it in the opposite direction. \\
